# Android (Play Store) metadata template

Suggested metadata to prepare before publishing:

- App name (short and full)
- Short description
- Full description
- High-res icon (512x512)
- Feature graphic (1024x500)
- Screenshots (several sizes, include phone and tablet if relevant)
- Category and tags
- Privacy policy URL
- Contact email and website
- Content rating
